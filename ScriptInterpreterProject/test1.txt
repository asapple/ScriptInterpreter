# 测试语法树生成测试one：符合语法规则的无逻辑冲突的语音脚本
Step welcome
Speak $name  您好，请问有什么可以帮您?
Listen 5 20
Branch 投诉 complainProc
Branch 账单 billProc
Branch 充值 refillProc
Silence silenceProc
Default defaultProc

Step complainProc
Speak 您的意见是我们改进工作的动力，请问您还有什么补充?
Listen 5, 50
Exit

Step billProc
Speak 您的本月账单是 + $amount + 元，感谢您的来电，再见
Exit

Step refillProc
Speak 请输出您的充值金额：
Default refillConfirmProc

Step refillConfirmProc
Speak 您的充值金额是 $refillAmount ，请确认（yes/on）
Branch yes refillThanks
Default refillProc

Step silenceProc
Speak 听不清，请您大声一点可以吗?
Listen 5 20
Branch 投诉 complainProc
Branch 账单 billProc
Branch 充值 refillProc
Silence silenceProc
Default defaultProc

Step defaultProc
Speak 没有明白您的意思，请您再说一次可以吗？
Listen 5 20
Branch 投诉 complainProc
Branch 账单 billProc
Branch 充值 refillProc
Silence silenceProc
Default defaultProc

Step refillThanks
Speak 已成功为您充值 $refillAmount ，感谢您的来电！
Exit
