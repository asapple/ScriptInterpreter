# 开发文档
## 需求分析
  **设计实现一个脚本分析解释器实现语音客服**
    - 处理用户的电话呼入
    - 借助媒体服务器播放提示音
    - 借助媒体服务器录音并将语音识别为文字
    - 借助自然语言分析引擎分析用户意图
    - 根据用户意图决定下一步播放的提示音

  **需求简化**
## 开发环境：
    - C Programming Language Standard: C++11
    - OS: Windows 10
    - IDE: Visual Studio 2019
